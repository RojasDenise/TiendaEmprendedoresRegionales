# Tienda de Emprendedores Regionales
Plataforma e-commerce regional diseñada para la digitalización de emprendimientos locales, facilitando la comercialización y visibilidad de sus productos.

## Estructura del Proyecto (Arquitectura en Capas)

* **/frontend**: Capa de Presentación desarrollada en React.
* **/backend**: Capa de Negocio y Aplicación desarrollada en Node.js.
* **/database**: Capa de Persistencia con scripts para SQL Server.
